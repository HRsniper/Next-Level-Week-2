## Instalação das dependências 

```sh
npm install 
```
ou
```sh
yarn
```
## Start

```sh
npm dev 
```
ou
```sh
yarn dev
```

## Criar Banco de Dados

```sh
npm dbdev 
```
ou
```sh
yarn dbdev
```
